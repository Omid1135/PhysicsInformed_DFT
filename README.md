# PhysicsInformed_DFT
Core code, data, and workflows for "Physics-Informed Modeling for Transferable Energetics in Substitutional Alloys." Contains 1,337 Fe/Mn and 96 Co/Ni DFT configurations, site-resolved descriptors, physics-informed ML models, and zero-/few-shot transfer analysis.
The above code and data  will be made publicly available after initial acceptance of the related article for publication.
